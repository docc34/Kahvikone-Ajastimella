Kahvikone Ajastimella

Projektin idea on aika simppeli. Halusin kahvia aina valmiiksi aamulla kun minulla on herätys. Projektissa oli useita kehitysvaiheita 4 digit 7 segment ruuduilla ja ledeillä, mutta olen tyytyväinen tähän versioon. Ainut parannettava laitteessa olisi kaiken virran saaminen kahvinkeittimestä. Tällä hetkellä arduino kyllä pyörii pelkällä kahvinkoneen virralla mutta kun keittäminen alkaa systeemi kyykkää ilman usb:stä saatua lisävirtaa.

Repossa on dokumentointi kahvinkeittimelle
